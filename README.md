# git-command-showcase 

## Clone Repository 
 git clone <repository-url> 

## Check status 
 git status 

## Stage Files 
 git add .

## Commit Changes 
 git commit -m "message" 

## Push Changes 
 git push 

## Pull Latest Changes 
 git pull 

## Create Branch
 git checkout -b branch-name

## Git Stash
 git stash
